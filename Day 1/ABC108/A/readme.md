[A-Pair](https://atcoder.jp/contests/ABC108/tasks/abc108_a)

# Mô tả
Cho một số nguyên K. Hãy tìm xem có bao nhiêu cặp số gồm 1 số chẵn và 1 số lẻ (1 <= 2 số đó <= K).
Lưu ý: cặp số (3, 2) và (2, 3) được tính là 1 cặp.

# Điều kiện 
2 ≤ K ≤ 100
K là số nguyên.

# Input:
1 số nguyên K.

# Output
số các cặp thỏa mãn.

# Ví dụ
| Input | Output |
| ----- | ----- |
| 3 | 2 |
| 6 | 9 |
| 11 | 30 |
| 50 | 625 |
